# AMOLE-SPORT

AMOLE-SPORT is a simple demo web application (HTML/CSS/JS) for sports betting UI and a small bet payout calculator. This repository now contains a lightweight single-page app that you can open locally or host via GitHub Pages.

## What I added
- index.html — landing page and UI
- assets/css/style.css — styles
- assets/js/app.js — simple bet calculator script
- Improved README

## Run locally
1. Clone the repo:
   git clone https://github.com/nesredinmohaa-creator/AMOLE-SPORT.git
2. Open `index.html` in your browser, or serve with a static server (recommended):
   - Python 3: `python -m http.server 8000` then open http://localhost:8000

## Deploy on GitHub Pages
1. In the repository settings -> Pages, set the source to the `main` branch and root folder.
2. Save — your site will be available at `https://nesredinmohaa-creator.github.io/AMOLE-SPORT/` after a few minutes.
